# Certificate Images Folder

This folder contains your certification, course completion, and achievement images.

## File Naming Convention:
- Use descriptive names: `udemy-react-course.png`, `aws-certification.jpg`
- Supported formats: PNG, JPG, JPEG
- Keep filenames short and web-friendly (no spaces, use hyphens)

## Image Requirements:
- Recommended size: 800x600px or similar aspect ratio
- Format: PNG or JPG
- Quality: High resolution for clear viewing in terminal

## Examples:
```
├── udemy-fullstack-course.png
├── deeplearning-ai-specialization.jpg  
├── hackathon-winner-2024.png
├── aws-solutions-architect.png
└── coursera-machine-learning.jpg
```

Add your images here and reference them in `/docs/achievements.json`
